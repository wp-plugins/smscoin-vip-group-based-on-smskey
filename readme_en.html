How it works:

This plugin enables you to provide paid access to any content on your website.
In reply SMS user receives a short text password (key),
and following its activation, user moves to VIP group. You define the page
on which user will pay for the transition to VIP group, indicate the following tag
[SMSCOINPAY]. You decide which articles you want close with the tags [rpayment].
You can emulate the process of paid registration by estblishing the period time in VIP group
(Vip Time) that equals to a large number, for example 99999 days. In this case, by paying for VIP status the user
has the right to access the hidden content during the enture time of website existence
At the end the time period in VIP group, the user
receives an SMS message the text of which is determined by you. (Tip: In the letter itself you should attach the link
to the payment page).

Installation:

  • You need to copy the folder "smscoin_rpayment" from the archive to a folder: wp-content/plugins


  • Go to the Admin panel of your "WordPress", Plugins section,
    You will find "SmsCoin VIP", click to activate.


  • Go to the menu "SmsCoin VIP", select "Settings" and
    follow the instructions for plugin configuration.

    * Enter the ID of your sms: key
    * Select the default language
    * Enter the secret code from sms:key settings
    * Enter a tag that hides the content (rpayment as a default value)
    usage example [rpayment] hidden text of your choice [/rpayment]
    * Enter the tag to display the page of the payment
    example of using [SMSCOINPAY], insert to any page of your choice
    * The time spent in VIP group, in days
    *The header of the letter
    * Letter



    • Example:


  • Set the write permission to the file
    /wp-content/plugins/smscoin_rpayment/data/local.js



  • Go to the menu "SmsCoin VIP" select "Rates"
    and click "Update Rates ".

    Installation is completed!


    Sms:key service setup process


  • In the control panel on smscoin.net website, go to sms:key service settings

  • Activate the option: Passwords' transfer.

  • Enter the URL of the handler: http://yoursite.com/wp-content/plugins/smscoin_rpayment/result_326.php

  • Enter the password for the signature request to handler, the password must be the same ,
    as in the module settings (any character set).

  • Time and the number of activations set 0.

    • Setup process is completed!

    Uninstallation:

  • Go to the Admin panel of your "WordPress", Plugins section,
    Find "SmsCoin VIP", click to deactivate

  • Delete the folder wp-content/plugins/smscoin_rpayment


    • Module is uninstalled.
